# Design Principles

## ビジュアル方針

- **Udemy/スタディサプリ** のコンテンツ構造をベースに
- **Apple的** な直感操作・洗練されたミニマルUI
- タブレットファースト設計（動画・問題の操作性重視）
- スマホでも快適に操作できるレスポンシブ対応

## UXルール

- スクリーンショット防止（Netflix方式のコンテンツ保護）
- 問題形式は3種: 選択式 / 記述式 / CBT方式
- 過去問ベースのコンテンツ
- 朝リマインド通知（5問）+ 夜の音声解説クイズ

## コンポーネント

- shadcn/ui をベースに拡張
- Geist Sans / Geist Mono フォント
- oklch カラーシステム（ダークモード対応）
- Lucide Icons を使用
